## TROUBLESHOOTING GUIDE

### Issue: Files Won't Upload

**Solution:**
- Check file size (Gumroad has limits for large files)
- Use ZIP file if uploading multiple files
- Try different browser
- Contact Gumroad support: help@gumroad.com

### Issue: Product Won't Appear

**Solution:**
- Make sure "Show on Gumroad" is checked
- Product may need 24 hours to appear in search
- Check your profile visibility settings

### Issue: Payment Not Going Through

**Solution:**
- Verify bank account is confirmed
- Minimum payout threshold ($10) might not be reached
- Check payment method is valid
- Contact support: support@gumroad.com

### Issue: Customer Can't Download Files

**Solution:**
- Send them the download link again
- Check file still exists in your product
- Make sure license key is correct (if enabled)
- Have them try different browser
- Resend from "Customers" page
